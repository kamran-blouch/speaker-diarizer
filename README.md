# Speaker Diarization

AI-powered speaker diarization tool that identifies who spoke when in your audio files.

## Features

- **Audio Upload**: Drag & drop or click to upload audio files (MP3, WAV, M4A, FLAC, etc.)
- **Speaker Identification**: AI automatically identifies and labels different speakers
- **Timeline Visualization**: Color-coded timeline showing speaker segments
- **Detailed Transcript**: Full transcript with speaker labels and timestamps
- **Export Options**: Download reports in multiple formats (HTML, TXT, SRT, JSON)

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Shadcn UI

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Usage

1. Upload an audio file with multiple speakers
2. (Optional) Set the expected number of speakers
3. Click "Analyze Speakers"
4. View the results with color-coded speaker segments
5. Download the report in your preferred format
